# Интернет-магазин парфюмов "Розовое Сияние"

Современный e-commerce сайт для продажи парфюмов в стиле candy/pink glow с использованием React, Vite и Supabase.

## Основные особенности

- Яркий, привлекательный дизайн в розово-фуксиевом стиле
- Адаптивная верстка для всех устройств
- Полнофункциональный каталог товаров с фильтрацией
- Страницы брендов, товаров и корзины
- Система авторизации пользователей
- Интеграция с Supabase backend

## Технологии

- React 18
- Vite
- React Router DOM
- Supabase (Auth, Database)
- CSS (без Tailwind или SCSS)

## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd perfume-store
```

2. Установите зависимости:
```bash
npm install
```

3. Создайте файл `.env` в корне проекта и добавьте конфигурацию Supabase:
```
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

4. Запустите приложение в режиме разработки:
```bash
npm run dev
```

5. Откройте в браузере http://localhost:5173
